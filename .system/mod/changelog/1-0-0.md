# KrepAPI 1.0.0

## Release

- **Stable 1.0 line**: first release using the **wire handshake** (magic `0x4B`, schema **1**, protocol semver **1.0.0** on `s2c_hello` / `c2s_client_info`). Older experimental builds (informally **a1.x.x**) used a different handshake and are **not** compatible.

## Features

- Shared binary protocol (`:protocol`) plus Fabric client (1.21.x and 26.1.x lines) and reference Paper plugin: **server-driven key bindings**, **raw key / mouse** with optional vanilla blocking, **intercept** slots, optional **handshake** and **minimum mod version** checks.
- **Docs** in-repo: wire protocol, client API, Paper plugin (including security notes for key packets).

## Compatibility

- **Minecraft (Fabric)**: **1.21.4–1.21.11** and **26.1–26.1.1** (separate JARs per game line).
- **Java**: 21 (1.21 mod), 25 (26.1 mod) per Gradle targets in this repo.
